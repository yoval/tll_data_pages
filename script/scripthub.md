# 脚本库

局域网地址：https://data.jello.top/tianlala_script_url

### 门店管理表格式化

导入门店管理表更新当日门店信息（每日仅可导一次，多次导入不会再更新），生成当日`增量门店管理表`。

注：

- 导入前需注意门店管理表是否完整。



### 渠道销售格式化

导入`美团管家`生成的报表，生成`各渠道销售统计`表。	

### 同比存量门店报表制作

选定日期进行对比，生成`各渠道同比销售情况`表。

### 常用原物料60日报货情况

通过中台导出文件制作常用原物料60日报货情况，类似的有[基础物料上次报货时间查询](/database/query?id=专题：基础物料上次报货时间查询)
